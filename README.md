# 🐒 Monkey Bot — Бот для прогрева аккаунтов WhatsApp и Max

## 📁 Структура проекта

```
monkey_bot/
├── bot.py              — основной файл бота
├── database.py         — работа с базой данных
├── keyboards.py        — клавиатуры и кнопки
├── requirements.txt    — зависимости Python
├── .env                — настройки (токен, ID владельца)
├── запустить.bat       — запуск одним кликом (Windows)
└── README.md           — это руководство
```

---

## ⚙️ Настройка

### 1. Создайте бота в Telegram
- Откройте [@BotFather](https://t.me/BotFather)
- Отправьте `/newbot`
- Укажите имя и username бота
- Скопируйте полученный **токен**

### 2. Узнайте свой Telegram ID
- Откройте [@userinfobot](https://t.me/userinfobot)
- Отправьте `/start`
- Скопируйте ваш **ID**

### 3. Заполните файл `.env`
```
BOT_TOKEN=1234567890:AAAbbbCCCdddEEEfffGGG
OWNER_ID=123456789
BOT_NAME=Monkey Bot
```

---

## 🚀 Запуск

### Windows (просто)
Дважды кликните на файл `запустить.bat`

### Вручную
```bash
pip install -r requirements.txt
python bot.py
```

## 👑 Команды владельца

| Команда | Описание |
|---------|----------|
| `/add_hours <user_id> <часы>` | Начислить часы пользователю |
| `/broadcast <текст>` | Рассылка всем пользователям |

---

## 🔄 Схема работы

1. Пользователь добавляет номер телефона
2. Выбирает настройки прогрева (срок, тип, параметры)
3. Нажимает **"Запустить прогрев"**
4. Владельцу приходит уведомление с **полными параметрами** и кнопкой
5. Владелец нажимает **"Отправить код"** и вводит код
6. Пользователь получает код прямо в боте

## 📊 База данных

База данных `bot_data.db` создаётся автоматически при первом запуске.
Хранит: аккаунты пользователей, их настройки и баланс часов.
